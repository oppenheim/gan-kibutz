# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project

This repository is for **גן קיבוץ - גן שיתופי בבית הכרם**, a cooperative kindergarten management project. The codebase is in early stages — no application code exists yet.

## Google Drive Integration

The project is connected to a shared Google Drive folder ("גן קיבוץ - גן שיתופי בבית הכרם", owned by omri@sbs.org.il). The `mcp__claude_ai_Google_Drive__search_files` tool is pre-authorized in `.claude/settings.local.json` and can be used to read documents and data from that folder.
